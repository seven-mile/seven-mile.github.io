奇妙多项式与求和综合：HDU6593

9月 14, 2020

Description

$n,q\le 3\times10^5$，要你对 $q$ 组询问 $(a,b,c,d)$ 回答

$$
\frac{d}{c+e^{ax+b}}
$$

在 $x_0=-b/a$ 处的泰勒展开 $(x-x_0)^n$ 项系数。

Analysis

作代换 $t=ax+b$，不难发现答案就是 $[t^n]a^n\cdot\frac{d}{c+e^t}$ ，那么 $a,b,d$ 都处理完了。
